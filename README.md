# Ruotu Project

## Overview
Ruotu is a personal productivity and lifestyle management app built with React and Vite. It includes modules for calendar, tasks, finance, savings, workout tracking, and settings.

## Features
- Calendar with events and birthdays
- Task management
- Finance tracking (expenses, incomes, savings goals)
- Workout logging and plans
- Quick notes and reminders
- Customizable themes

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
